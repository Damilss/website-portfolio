Place your licensed Aeonik Pro font files in this folder with these names:

- `AeonikPro-Regular.woff2` (required)
- `AeonikPro-Medium.woff2` (recommended)
- `AeonikPro-Bold.woff2` (recommended)

Optional legacy fallbacks:

- `AeonikPro-Regular.woff`
- `AeonikPro-Medium.woff`
- `AeonikPro-Bold.woff`

The site is already configured to load these files via `@font-face` in `app/globals.css`.
