# CSC 202 — Data Structures

> The data structures course — lists, trees, graphs, and the algorithms over them.

CSC 202 is Cal Poly's data structures course: the structures that back real
programs — dynamic arrays, linked lists, trees, hash maps, graphs — and the
algorithms that operate on them, worked through across a series of projects.
This repo collects that coursework.

One later project covered graph algorithms — here is a topological sort of a
directed acyclic graph, done with Kahn's algorithm:

```python
def tsort(edge_list: list[list[str]]) -> list[str]:
    """Topological sort of a DAG — returns vertices in dependency order."""
    adj, in_degrees, vertices = defaultdict(list), defaultdict(int), set()

    for u, v in edge_list:
        adj[u].append(v)
        in_degrees[v] += 1
        vertices.update([u, v])

    # start from every vertex that nothing points to
    stack = [v for v in vertices if in_degrees[v] == 0]
    result = []

    while stack:
        vertex = stack.pop()
        result.append(vertex)
        for next_vertex in adj[vertex]:
            in_degrees[next_vertex] -= 1
            if in_degrees[next_vertex] == 0:
                stack.append(next_vertex)

    if len(result) != len(vertices):
        raise ValueError("input contains a cycle")
    return result
```

*Kahn's algorithm — repeatedly take a vertex with no remaining dependencies, then relax its edges.*
