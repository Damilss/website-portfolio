# CSC 203 — Lab 2

> First steps in object-oriented Java — defining classes and fields.

An early CSC 203 lab: three short exercises across one week, getting
comfortable with the basics of object-oriented Java — declaring classes,
fields, and the shape of an object before any behavior goes on it.

```java
package task_1;

public class Dog {
    String name;
    String breed;
}
```

*Week-one OOP — a class as a plain bundle of state, before methods and encapsulation.*
