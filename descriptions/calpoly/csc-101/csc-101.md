# CSC 101 — Fundamentals of Computer Science

> The first CS course at Cal Poly — the fundamentals, in Python.

CSC 101 is Cal Poly's introduction to computer science: the core building
blocks — variables, control flow, functions, strings, lists — worked through in
Python across a series of labs and programming assignments. This repo collects
that coursework.

```python
def vowel_count(x: str) -> int:
    vowels = 0
    n = list(x)
    for i in n:
        if i in 'aeiouAEIOU':
            vowels += 1
    return vowels
```

*An early assignment function — counting vowels in a string, one of the first programs in the course.*
