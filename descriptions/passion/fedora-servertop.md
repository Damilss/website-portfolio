# fedora-servertop

> A home server running Fedora — owning my own infrastructure end to end.

fedora-servertop is my homelab: a self-hosted Fedora machine that I run as both
a server and a desktop. The goal is simple — own everything. Instead of renting
storage, compute, and services from someone else, run them myself, learn how
the pieces actually fit together, and document the journey of standing it all
up and keeping it alive.

## Status

Ongoing — a personal infrastructure project, kept in version control as it grows.
