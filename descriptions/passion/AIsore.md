# AIsore

[aisoreapp.com](https://aisoreapp.com)

AI content detection platform. Analyze text, photos, videos, and links to determine whether content was generated by AI or created by a human.

---

## Overview

AIsore uses the Hive AI API to score content on a human-to-AI spectrum. Users submit content through the mobile app, detection jobs are processed asynchronously, and results are returned with a verdict (Likely AI, Likely Human, or Uncertain) along with a confidence score and suspected model.

The platform runs on a credit-based system:

| Content Type | Credit Cost |
|--------------|-------------|
| Text         | 2           |
| Photo        | 4           |
| Video        | 10          |
| Link         | 10          |

Plans: Free (30 credits/month), Pro (350 credits/month, $4.99/mo).

## Architecture

```
mobile (React Native / Expo)
    |
    v
backend (Express API)
    |
    ├── PostgreSQL (users, analyses, jobs)
    ├── Redis + BullMQ (async job queue)
    ├── Hive AI (detection API)
    └── DigitalOcean Spaces (file storage)
```

---

## Tech Stack

**Backend**
- Node.js, Express 5, TypeScript
- PostgreSQL, Prisma ORM
- BullMQ, Redis
- JWT authentication with refresh token rotation
- Zod validation

**Mobile**
- React Native 0.76, Expo 52
- Expo Router (file-based routing)
- TypeScript
- iOS Share Extension for detecting content shared from other apps

## Setup

### Backend

```bash
cd backend
npm install
cp .env.example .env   # fill in your values
npm run migrate
npm run dev
```

Required environment variables: `DATABASE_URL`, `REDIS_URL`, `JWT_SECRET`, `HIVE_API_KEY`, `DO_SPACES_KEY`, `DO_SPACES_SECRET`, `DO_SPACES_BUCKET`, `DO_SPACES_ENDPOINT`.

### Mobile

```bash
cd mobile
npm install
```

Set `EXPO_PUBLIC_API_URL` in `mobile/.env` to your backend URL, then:

```bash
npx expo run:ios --device   # physical device
npx expo start              # development with simulator
```

## API Routes

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/auth/signup` | Create account |
| POST | `/api/auth/signin` | Login |
| POST | `/api/auth/refresh` | Refresh access token |
| POST | `/api/detect/text` | Submit text for detection |
| POST | `/api/detect/photo` | Upload photo for detection |
| POST | `/api/detect/video` | Upload video for detection |
| POST | `/api/detect/link` | Submit URL for detection |
| GET | `/api/detect/status/:jobId` | Poll job result |
| GET | `/api/history` | Paginated analysis history |
| GET | `/api/user/profile` | Current user info and credits |
