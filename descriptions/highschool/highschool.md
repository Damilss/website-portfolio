# High School Portfolio

> Where it started — a portfolio site built in AP Computer Science Principles.

This is the oldest thing here: a personal portfolio site I built in 2022 for AP
Computer Science Principles, dating back to sophomore year of high school. It
collects the projects from that class — small games, coding exercises, and
design work — and it's still live on GitHub Pages.

![A Mad Libs project from the portfolio](/work-assets/highschool/madlibs.jpg)

## About the build

It's plain HTML, JavaScript, and CSS — no frameworks, no dependencies — written
back when a mix of GitHub and Replit was the whole toolchain. It's deliberately
left as-is: a snapshot of where the work started, before any of the rest of
this portfolio existed.

## Tech stack

- HTML, JavaScript, CSS — no dependencies
- Deployed on GitHub Pages

## Status

Archived — a 2022 time capsule, kept exactly as it was.
