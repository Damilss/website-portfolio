# Client Work

Freelance and contract web work — sites I've designed and built for paying
clients. The focus is the same every time: clean, fast, professional sites that
the client can actually maintain.

## What's in here

- **lilianal.com** — a personal website built for a client with Next.js,
  TypeScript, and Tailwind CSS. Frontend-only, deployed on Vercel.
- **westcoastbeautyco.com** — a website for a beauty and wellness brand. Next.js
  + TypeScript + Tailwind, with a Resend-powered contact form and an Instagram
  feed pulled from a public JSON endpoint.
