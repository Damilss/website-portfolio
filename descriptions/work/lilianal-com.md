# lilianal.com

> A personal website built for a client — clean, fast, easy to hand off.

lilianal.com is a freelance project: a professional personal website built for
a client with Next.js, TypeScript, and Tailwind CSS. It's a frontend-only build
— no backend needed — with the priorities being performance, accessibility,
mobile responsiveness, and being genuinely easy for the client to maintain
afterward.

## Scope

- A landing page, an about section, and a contact section (form or direct links)
- An SEO baseline — metadata, sitemap, and analytics where the client wants it
- Light, smooth motion via Framer Motion
- Deployed on Vercel, with the domain managed through Cloudflare

## Tech stack

- Next.js (App Router), React, TypeScript
- Tailwind CSS, Framer Motion
- Vercel hosting, Cloudflare DNS

## Status

Active — initial setup and production build underway; client content lands as it is approved.
